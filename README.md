# Teen Mental Health Depression Prediction Model

## Project Overview
This project predicts depression risk among teenagers using machine learning.

## Objective
To identify mental health risks early using behavioral and emotional indicators.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Data Analysis
4. Model Training
5. Prediction
6. Evaluation

## Files in this Repository
- notebooks → Google Colab notebook
- data → Dataset
- images → Graphs and screenshots

## Results
The model predicts teen depression risk using machine learning techniques.
## Models Used
- K-Nearest Neighbors (KNN)
- Naive Bayes

## Model Performance

### KNN Model
Accuracy: 96.94%

Classification Report:

| Metric | Class 0 | Class 1 |
|---------|---------|---------|
| Precision | 0.98 | 0.25 |
| Recall | 0.99 | 0.11 |
| F1-score | 0.98 | 0.15 |

### Naive Bayes Model
Accuracy: 96.94%

Classification Report:

| Metric | Class 0 | Class 1 |
|---------|---------|---------|
| Precision | 0.98 | 0.25 |
| Recall | 0.99 | 0.11 |
| F1-score | 0.98 | 0.15 |

## Key Insight
The dataset was imbalanced, which resulted in high overall accuracy but lower prediction performance for minority depression cases. Future improvements may include balancing techniques such as SMOTE and hyperparameter tuning.
