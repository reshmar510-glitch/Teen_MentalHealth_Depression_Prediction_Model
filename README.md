# Teen Mental Health Depression Prediction Model

## Project Overview
This project predicts depression risk among teenagers using machine learning.

## Objective
To identify mental health risks early using behavioral and emotional indicators.

## Technologies Used
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Project Workflow
1. Data Collection
2. Data Cleaning
3. Data Analysis
4. Model Training
5. Prediction
6. Evaluation

## Files in this Repository
- notebooks → Google Colab notebook
- data → Dataset
- images → Graphs and screenshots

## Results
The model predicts teen depression risk using machine learning techniques.
